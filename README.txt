COMO USAR O PROJETO

1. Extraia o arquivo ZIP.
2. Abra o VS Code.
3. Vá em File > Open Folder e selecione a pasta site_gabriell_vitoria.
4. Instale a extensão Live Server no VS Code.
5. Clique com o botão direito em index.html.
6. Clique em Open with Live Server.

COMO TROCAR AS IMAGENS

Coloque suas fotos dentro da pasta assets e use estes nomes:

- capa.jpg
- memoria1.jpg
- memoria2.jpg

Depois, no arquivo index.html, troque:

assets/capa.svg por assets/capa.jpg
assets/memoria1.svg por assets/memoria1.jpg
assets/memoria2.svg por assets/memoria2.jpg

COMO COLOCAR A MÚSICA

Coloque uma música MP3 dentro da pasta assets com o nome:

musica.mp3
